# SPV Structure — HoldCo

## SunFarm Holdings, S.R.L.

### Purpose

HoldCo is the master holding entity. It owns 100% of all operating subsidiaries. No project-level assets, contracts, or liabilities sit at HoldCo. Its sole function is structural governance, capital allocation, and consolidated reporting.

### Ownership

| Subsidiary | Ownership | Asset Class |
|---|---|---|
| LandCo (SunFarm Land, S.R.L.) | 100% | Real property, land leases |
| Energy SPV (SunFarm Energy, S.R.L.) | 100% | Concession, PPA, BESS, carbon rights |
| Token SPV (SunFarm Digital, S.R.L.) | 100% | Revenue-share token issuance |

### Governance

- Board: Minimum 3 directors (1 independent)
- Reserved matters: Asset disposal >$1M, new debt, SPV creation, distributions
- Reporting: Quarterly consolidated financials, annual audit
- Domicile: Dominican Republic (mirroring operating jurisdiction)

### Capital Flow

```
External Capital → HoldCo → Downstream equity injections
                          ← Upstream dividend distributions
                          ← Management fees (arm's length)
```

### Key Restrictions

- HoldCo does NOT hold project-level permits or contracts
- HoldCo does NOT guarantee project debt (non-recourse to subs)
- HoldCo does NOT issue tokens (Token SPV only)
- HoldCo does NOT hold land title (LandCo only)
